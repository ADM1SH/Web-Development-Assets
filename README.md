# Web Development Assets & Learning Materials

## Overview
A curated collection of assets, lecture slides, and reference materials for mastering web development. This repository serves as a personal knowledge base for front-end and back-side technologies.

## Contents
- **Lecture Materials**: Detailed PPTX slides covering HTML, CSS, JavaScript, and PHP.
- **Lab Resources**: Comprehensive PDF guides and starter code for various web development tasks.
- **Project Specifications**: Guidelines and templates for major course projects.

## Usage
These materials are intended for study and reference. They provide a structured path from basic internet concepts to building functional web applications.
